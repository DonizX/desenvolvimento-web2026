# 📝 Resumo da Aula 04 - Desenvolvimento Web

## 🌐 Registro de Domínio

O **registro de domínio** é o processo de adquirir um nome único para um site na internet, como por exemplo:

www.meusite.com

Esse nome é o endereço utilizado pelos usuários para acessar o site, sendo essencial para sua identificação online.

---

## 🖥️ Hospedagem de Sites

A **hospedagem de sites** é o serviço responsável por armazenar os arquivos de um site em um servidor, permitindo que ele fique disponível na internet.

Sem hospedagem, o site não pode ser acessado pelos usuários.

---

## 🧩 Sistema de Gerenciamento de Conteúdo (CMS)

Um **CMS (Content Management System)** é uma ferramenta que permite criar, editar e gerenciar conteúdos de um site sem a necessidade de programar.

Com um CMS, é possível:

* Criar páginas
* Publicar conteúdos
* Gerenciar usuários
* Personalizar o site

---

## 🧱 Wordpress e Principais Funcionalidades

O **WordPress** é um dos CMS mais utilizados no mundo.

Principais funcionalidades:

* Criação de sites e blogs
* Uso de temas (design do site)
* Instalação de plugins (funcionalidades extras)
* Gerenciamento de conteúdo de forma simples

Ele facilita o desenvolvimento mesmo para quem não possui muito conhecimento técnico.

---

## 🛠️ Ferramentas para Configuração do Wordpress

Para configurar o WordPress, são utilizadas ferramentas como:

* Painel de hospedagem (cPanel, por exemplo)
* Instaladores automáticos
* Banco de dados (MySQL)
* Acesso FTP (para envio de arquivos)

Essas ferramentas permitem instalar, configurar e manter o site funcionando corretamente.

---

## 🔄 Timeline de um Site

O desenvolvimento de um site segue um fluxo organizado:

IDEIA → DOMÍNIO → SERVIDOR (HOSPEDAGEM) → DESENVOLVIMENTO → DEPLOY → SITE ONLINE

### 📌 Explicação do fluxo

* **Ideia:** planejamento do projeto
* **Domínio:** escolha do nome do site
* **Hospedagem:** onde o site será armazenado
* **Desenvolvimento:** criação do site
* **Deploy:** publicação do site
* **Site Online:** disponível para acesso
