# 📝 Resumo da Aula 05 - Introdução ao JavaScript (Vanilla JS)

## 🌐 HyperText Markup Language (HTML)

O **HTML (HyperText Markup Language)** é a linguagem de marcação utilizada para estruturar o conteúdo de páginas web.

Com o HTML, é possível definir elementos como:

* Títulos
* Parágrafos
* Links
* Imagens
* Listas

Ele é a base de qualquer site, responsável pela estrutura da página.

---

## 🎬 Mídias em HTML

O HTML permite a inserção de diferentes tipos de mídia em uma página web.

Principais exemplos:

* Imagens (`<img>`)
* Vídeos (`<video>`)
* Áudios (`<audio>`)

Esses recursos tornam o site mais interativo e dinâmico.

---

## 🔍 SEO em HTML

O **SEO (Search Engine Optimization)** é um conjunto de práticas para melhorar o posicionamento de um site nos mecanismos de busca.

No HTML, isso envolve:

* Uso correto de tags semânticas
* Definição de títulos (`<title>`)
* Uso de meta tags
* Estrutura organizada do conteúdo

Um bom SEO aumenta a visibilidade do site na internet.

---

## 🎨 Cascading Style Sheets (CSS)

O **CSS (Cascading Style Sheets)** é utilizado para estilizar páginas web.

Com o CSS, é possível:

* Definir cores
* Alterar fontes
* Ajustar espaçamentos
* Criar layouts

Ele é responsável pela aparência visual do site.

---

## ⚙️ JavaScript (Vanilla JS)

O **JavaScript** é uma linguagem de programação utilizada para adicionar interatividade às páginas web.

O termo **Vanilla JS** refere-se ao uso do JavaScript puro, sem bibliotecas ou frameworks.

Com JavaScript, é possível:

* Manipular elementos da página (DOM)
* Criar interações com o usuário
* Validar formulários
* Atualizar conteúdos dinamicamente

---

## 💭 Considerações sobre a aula

A aula apresentou os fundamentos das tecnologias essenciais para o desenvolvimento web: HTML, CSS e JavaScript. Foi possível compreender o papel de cada uma delas na construção de um site, desde a estrutura, passando pelo estilo, até a interatividade, formando a base para o desenvolvimento de aplicações web modernas.
